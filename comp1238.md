# COMP1238 Notes

## Week 1 - Computer History
- Early computers were large, expensive, and used for special purposes.
- Modern computers are smaller, faster, and more powerful than early computers.

## Week 2 - Bits and Bytes
- A bit is the smallest unit of data and can be 0 or 1.
- A byte is 8 bits and is used to represent characters and data.

## Week 3 - Data Representation
- ASCII is a system used to represent characters using numbers.
- Computers store all data (text, images, sound) as binary.

## Week 4 - Command Line and Interfaces
- CLI (Command Line Interface) uses text commands to control the computer.
- GUI (Graphical User Interface) uses icons, windows, and mouse.

## Week 5 - Git and GitHub
- Git is used to track changes in files and manage versions.
- GitHub is used to store repositories and share code online.
